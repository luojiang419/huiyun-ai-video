# Grsai 模型状态检测

## API端点

```
GET https://grsai.dakka.com.cn/client/common/getModelStatus?model={modelName}
```

## 请求参数

- `model`: 模型名称（URL参数）

## 响应格式

```json
{
  "code": 0,
  "data": {
    "status": true,
    "error": ""
  },
  "msg": "success"
}
```

## 字段说明

- `status`: 模型状态（true=可用，false=不可用）
- `error`: 错误信息（为空表示无错误）

## 使用示例

### curl

```bash
curl -X GET "https://grsai.dakka.com.cn/client/common/getModelStatus?model=nano-banana-pro"
```

### Python

```python
import requests

def check_model_status(model_name):
    url = f"https://grsai.dakka.com.cn/client/common/getModelStatus?model={model_name}"
    response = requests.get(url)
    result = response.json()
    if result["code"] == 0:
        return result["data"]["status"]
    return None
```

### 批量检测

```bash
for model in sora-image gpt-image-1.5 nano-banana-pro; do
  echo "Testing: $model"
  curl -s "https://grsai.dakka.com.cn/client/common/getModelStatus?model=$model" | grep -o '"status":[^,}]*'
done
```
