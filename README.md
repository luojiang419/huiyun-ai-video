# 绘云 AI 工作区

这个仓库用于管理 `G:\data\app\AI` 工作区中的桌面应用源码与相关技术文档。

## 主要目录

- `flutter_grsai_video_gen/`：Flutter Windows 桌面应用主工程
- `技术文档/`、`api文档/`、`DOCS/`：开发与接口资料
- `AI规则/`：提示词与规则配置

## 发布说明

- 源码仓库：`luojiang419/huiyun-ai-video`
