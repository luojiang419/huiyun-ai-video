# 绘云AI v2.0 开发者文档

## 项目概述

**项目名称**: flutter_grsai_video_gen
**版本**: 2.0.0+1
**描述**: GRSAI视频生成器 - Flutter版本
**开发者**: Leo.j

这是一个基于Flutter开发的桌面应用程序，主要用于AI图片生成、影视分镜制作和资产管理。

---

## 技术栈

### 核心框架
- **Flutter SDK**: ^3.10.4
- **状态管理**: flutter_riverpod ^2.4.0
- **网络请求**: dio ^5.4.0

### 主要依赖
- **文件操作**: path_provider ^2.1.0, file_picker ^6.1.0
- **本地存储**: shared_preferences ^2.2.0
- **图片处理**: cached_network_image ^3.3.0, photo_view ^0.14.0
- **桌面支持**: window_manager ^0.3.0, desktop_drop ^0.4.0
- **工具库**: uuid ^4.0.0, intl ^0.18.0, url_launcher ^6.2.0

### 4. AI剧本资产提取 (ScriptAssetLibraryScreen)

**文件位置**: `lib/screens/script_asset_library_screen.dart`

**功能**: 使用AI自动分析剧本，提取人物造型、场景、道具等视觉资产

**核心功能**:

#### 4.1 剧本资产提取
- **输入**: 完整剧本文本
- **AI分析**: 使用Chat API进行智能拆解
- **输出**: 按场次组织的资产列表（人物造型、场景、道具）

#### 4.2 资产类型
- **character**: 人物造型（服装、发型、妆容、状态）
- **scene**: 场景（环境、氛围、光影）
- **prop**: 道具（物品、车辆、生物）

#### 4.3 时空连续性分析
AI会自动追踪人物造型的变化：
- 时间推移（早晨睡衣 → 上午西装 → 深夜湿透西装）
- 环境影响（雨淋、打斗、污渍、伤痕）
- 场合变化（家/公司/战场）

#### 4.4 批量生图功能
- 选择需要生成的资产
- 使用提取的描述作为提示词
- 批量生成资产参考图
- 自动保存到资产库

**涉及服务**:
- `AssetExtractionService.extractAssets()`: 提取资产
- `ApiService.chatStream()`: 流式对话
- `GenerateLogicService.generate()`: 生成图片

**系统提示词位置**: `data/Settings/asset_extraction_system_prompt.txt`

**数据模型**:
- `ScriptAssetExtraction`: 场次资产提取结果
- `ExtractedAsset`: 单个提取的资产

**核心代码**:
- 提取逻辑: [asset_extraction_service.dart:113-162](lib/services/asset_extraction_service.dart#L113-L162)
- 批量生图: [script_asset_library_screen.dart](lib/screens/script_asset_library_screen.dart)

## 常见问题

### Q1: 生成图片失败
**检查项**:
1. API配置是否正确
2. 网络连接是否正常
3. 积分是否充足
4. 模型是否在线（查看模型状态指示灯）

### Q2: 会话数据丢失
**原因**: 会话数据保存在 `data/sessions/` 目录
**解决**: 检查文件是否存在，权限是否正确

### Q3: 参考图片无法上传
**检查项**:
1. 图片格式是否支持（png, jpg, jpeg, webp）
2. `data/input/` 目录是否存在
3. 文件权限是否正确
